The next element is the slider. When you pull the slider a yellow rectangle will appear and will grow or shrink
depending on which direction you move the slider to.
The most important thing I have learned at this stage is how to clear a canvas.
Each time you change the size of the rectangle the old one will be deleted and a new one is created.
Otherwise the rectangles would be drawn on top of each other and you could only see a difference
if you would create a bigger rectangle than before.

The first more challenging part from the course was to build this filter function.
You can upload an image and choose between the different filters. However the first row of filters handels every image equally.
But i was not very satisfied with that because when you wanted to greyscale a white image, nothing changed
or when you used a red filter on an image with a green background, the background turned yellow.
To solve this problem I created the second row of the filters. I tried out different methods to change the colorvalues the way I wanted them to be.
The result is that I set borders for the colortones and depending on how i wanted a filter to work a different function is called.



choose a filter






The last task was to create this filter which merges two images.
It scans the colorvalues from each pixel from the first image and if the greenvalue is bigger than the redvalue and the bluevalue
combined the program replaces it with a pixel from the second image. To make sure that this filter would work as expected
I had to set the two images to the same size so that the amount of pixels are identical.
